## установка зависимостей проекта

```sh
npm install
```

### запуск проекта в режиме разработки

```sh
npm run dev
```

### сборка проекта для продакшена

```sh
npm run build
```

### запуск тестов

```sh
npm run test:unit
```

### линтинг

```sh
npm run lint
```
